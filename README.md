# Excel Cleaner Web App

## 📌 Project Description
This project is a Flask-based web application that cleans messy Excel data into a structured format.

## 🚀 Features
- Removes empty rows & columns
- Standardizes column names
- Cleans missing values
- Converts date formats (DD/MM/YYYY)
- Generates unique employee IDs
- Fills missing salary & age values

## 🛠 Tech Stack
- Python (Flask)
- Pandas
- HTML, CSS
